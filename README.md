# 🎮 小精靈 Pac-Man

經典小精靈遊戲，支援 Mac 與 Windows，**不需要網路，雙擊就能玩！**

---

## 📥 下載方式

點右上角綠色 **Code** 按鈕 → **Download ZIP** → 解壓縮

---

## 🚀 啟動方式

### Mac 用戶
1. 解壓縮後，雙擊「**Mac啟動遊戲.command**」
2. 如果出現「無法打開」：右鍵 → 打開 → 打開
3. 第一次會自動安裝 pygame（約 30 秒），之後直接秒開

### Windows 用戶
1. 解壓縮後，雙擊「**Windows啟動遊戲.bat**」
2. 如果出現藍色安全警告：點「更多資訊」→「仍要執行」
3. 第一次會自動安裝 pygame（約 30 秒），之後直接秒開

> ⚠️ 需要先安裝 Python 3：https://www.python.org/downloads/
> 安裝時記得勾選「Add Python to PATH」

---

## 🕹️ 操作說明

| 按鍵 | 功能 |
|------|------|
| ← → ↑ ↓ | 移動小精靈 |
| R | 重新開始 |
| ESC | 離開遊戲 |
| 任意鍵 | 開始 / 繼續 |

---

## 🎯 遊戲規則

- 吃掉地圖上所有的**小白點**即過關
- 吃到**大白點**後，鬼魂會變藍色，可以反吃鬼魂得分
- 被鬼魂碰到會失去一條命，共 **3 條命**
- 挑戰看看能破幾關！
