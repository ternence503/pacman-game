## 💾 方法二：下載到電腦玩（沒有網路也能玩）

### 第一步：下載遊戲

點這個頁面右上角的綠色 **`Code`** 按鈕，選 **`Download ZIP`**

![下載示意](https://docs.github.com/assets/cb-47677/mw-1440/images/help/repository/code-button.webp)

### 第二步：解壓縮

找到剛剛下載的 `pacman-game-main.zip`，**雙擊**它，會自動解壓縮變成一個資料夾。

### 第三步：打開遊戲

進入解壓縮後的資料夾，找到 **`index.html`**，**雙擊**它！

瀏覽器會自動打開，然後就可以玩了 🎮

> 💡 **不管是 Mac 還是 Windows 都是一樣的步驟！**

## 🎯 遊戲規則

- 吃掉地圖上**所有小白點**就過關！
- 吃到**大白點**➡️ 鬼魂變藍色 ➡️ 可以反過來吃鬼魂！
- 被鬼魂碰到會扣一條命，共有 **3 條命**
- 每過一關速度變快，看你能闖幾關！

---

## 🔊 音效

遊戲有內建音效，右下角按 **🔊** 可以開關靜音。

---

## ❓ 打開 index.html 沒有聲音？

請改用 **Chrome** 或 **Edge** 瀏覽器打開，音效最完整。

<summary>🐍 進階：Python 版（需要安裝 Python）</summary>

如果你會使用 Python，`Python版/` 資料夾裡有原始 pygame 版本：

1. 先安裝 Python 3：https://www.python.org/downloads/（安裝時勾選「Add Python to PATH」）
2. Mac：雙擊 `Python版/Mac啟動遊戲.command`
3. Windows：雙擊 `Python版/Windows啟動遊戲.bat`
4. 第一次會自動安裝 pygame（約 30 秒），之後直接秒開
