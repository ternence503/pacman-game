# 🟡 小精靈 PAC-MAN

> 純瀏覽器小精靈遊戲，無需安裝任何軟體，下載即玩！

---

## 🎮 線上直接玩（最簡單）

點這個連結，用瀏覽器開啟就能玩：

👉 **https://ternence503.github.io/pacman-game/**

---

## 💾 下載到電腦玩（離線也能玩）

### 第一步：下載遊戲

點右上角綠色的 **`Code`** 按鈕 → 選 **`Download ZIP`**

### 第二步：解壓縮

找到下載的 `pacman-game-main.zip`，雙擊解壓縮。

### 第三步：啟動遊戲

| 系統 | 做法 |
|------|------|
| **Mac** | 雙擊 `啟動遊戲_Mac.command`<br>（第一次開啟：右鍵 → 打開 → 打開） |
| **Windows** | 雙擊 `啟動遊戲_Win.bat` |
| **其他** | 用任何瀏覽器直接開啟 `index.html` |

## 🎯 遊戲規則

- 吃掉地圖上所有的**小白點**即過關
- 吃到**大白點**後，鬼魂變藍色，可以反吃鬼魂得分
- 被鬼魂碰到會失去一條命，共 **3 條命**
- 每過一關速度加快，挑戰看看能破幾關！

---

## 👻 鬼魂介紹

| 鬼魂 | 顏色 | 行為 |
|------|------|------|
| Blinky | 🔴 紅 | 直接追蹤小精靈 |
| Pinky  | 🩷 粉 | 預測小精靈前方4格 |
| Inky   | 🩵 青 | 搭配 Blinky 夾攻 |
| Clyde  | 🟠 橘 | 遠追近逃 |

遊戲內建即時合成音效（Web Audio API），右下角 **🔊** 按鈕可切換靜音。

## 🐍 Python 版（進階）

如果偏好在桌面應用程式執行，`Python版/` 資料夾內有原始 Python/pygame 版本：

| 系統 | 做法 |
|------|------|
| **Mac** | 雙擊 `Python版/Mac啟動遊戲.command`（需先安裝 Python 3） |
| **Windows** | 雙擊 `Python版/Windows啟動遊戲.bat`（需先安裝 Python 3） |

> ⚠️ 需要先安裝 Python 3：https://www.python.org/downloads/
> 安裝時記得勾選「Add Python to PATH」
> 第一次啟動會自動安裝 pygame（約 30 秒），之後直接秒開

| 版本 | 需求 |
|------|------|
