【前置需求】
  需要安裝 Python 3（免費）
  下載網址：https://www.python.org/downloads/
  ⚠️  Windows 安裝時記得勾選「Add Python to PATH」

【啟動方式】
  Mac     → 雙擊「Mac啟動遊戲.command」
  Windows → 雙擊「Windows啟動遊戲.bat」

  首次啟動會自動安裝 pygame、numpy（需要網路）
  之後就可以完全離線遊玩

【遊戲操作】
  ↑ ↓ ← →   移動小精靈
  R           重新開始
  ESC         離開遊戲

【遊戲規則】
  • 吃掉所有豆子即可過關
  • 吃到大顆能量豆，鬼魂會變藍色（可以反吃！）
  • 連續吃鬼得分加倍：200 → 400 → 800 → 1600
  • 共 3 條命，被鬼碰到會扣命
  • 難度隨關卡提升

【檔案說明】
  遊戲主程式.py      → 遊戲本體（不要刪除）
  Mac啟動遊戲.command → Mac 一鍵啟動腳本
  Windows啟動遊戲.bat → Windows 一鍵啟動腳本
  使用說明.txt        → 本說明文件
